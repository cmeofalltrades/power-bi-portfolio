# IT Support Ticket Dashboard

## Objective
Analyze IT support tickets to identify trends in volume, priority, and resolution time.

## Dataset
- Simulated IT support ticket data
- Users and devices modeled as dimensions

## Key Metrics
- Total Tickets
- Open vs Closed Tickets
- Average Resolution Time
- Tickets by Category and Priority

## Features
- Interactive slicers
- KPI cards
- Star schema data model
